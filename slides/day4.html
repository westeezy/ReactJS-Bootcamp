Controlled vs Uncontrolled Inputs

  • A form component with a value set is known as a 
    controlled input
  • A form component without a value set is known as an 
    uncontrolled input

Controlled Input

  • Always shows whats in it's value property
  • Thus direct user input has no effect
  • onChange and state can be used to change it's value
  • React owns state of the component at any time

Examples

getInitialState() {
  return { value: "Person" };
}

handleChange(event) {
  this.setState({ value: event.target.value });
}

render() {
  return (
  

    
    {'Hello ' + this.state.value}
  
);
}

Uncontrolled Input

  • Input that does not have a value prop or value is set to null
  • Value reflects user input
  • can still use onChange if you want to listen to value updates

Examples


handleChange(event) {
  let newValue = event.target.value;
  this.setState({ value: newValue });
}
render() {
  return (

            
            {'Hello ' + this.state.value}
          
);
}
